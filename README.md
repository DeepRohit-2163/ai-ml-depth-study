## Features

- **Depth badges** — Surface / Mid / Depth Probe / Killer labels on every concept
- **Interview Q&A** — toggle-able answers for self-quizzing
- **Progress tracking** — topic mastery and study plan checkboxes saved in localStorage
- **Global search** — `Ctrl+K` to search across all content
- **Scroll-spy sidebar** — jump to any section instantly
- **Offline-capable** — pure HTML/CSS/JS, no build step, no dependencies

## Stack

Static HTML + CSS + vanilla JS. Zero frameworks. Deployed via GitHub Actions → GitHub Pages.

## Local use

Just open `index.html` in a browser — no server needed.
