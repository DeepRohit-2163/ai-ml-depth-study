⚠ The Honest Rules

You don't fail because you don't know enough topics. You fail because you answer at API level when the interviewer probes one level deeper.

Rule: For every topic below, don't just read — close the page and explain it out loud as if the interviewer just asked "okay but WHY does that work?" If you can't answer that follow-up from memory, you haven't learned it yet. The "Interrogation" section on each topic page is the actual exam.

How to use this plan

01

Read → Explain → Drill

For each day: read the topic page, then close it and explain the concept out loud. Then open the Interrogation section and answer every question before revealing the answer.

02

Mark only when you can explain

Use the "Mark as Mastered" button on each topic page only when you can answer the depth-probe questions without looking. Not when you feel you've read enough.

03

Don't skip Phase 1

The math/classical ML topics are tested even in GenAI-heavy rounds at Amazon and Deloitte. Candidates who skip these get caught on Entropy/Gini calculations and L1 vs L2 derivations.

04

Days 57-60 are non-negotiable

Mock interrogation mode. Pick any topic, open a blank doc, and write out the full depth-probe answer chain as if an Amazon bar-raiser is asking. No cheating.

Phase 7
Mock Interrogation — The Final Exam
Days 58–60 · This is the most important phase. Don't skip it.

How to run a mock interrogation session

1. Open a blank doc or grab a piece of paper.
2. Pick a topic at random from any pillar.
3. Imagine an Amazon bar-raiser is sitting across from you. They ask: "Walk me through [topic]."
4. Write or speak the full answer — intuition, math, trade-offs, when it breaks.
5. They push back: "Okay but WHY does [mechanism] work?" Answer without notes.
6. Check against the Interrogation section. Mark any gap.
7. Repeat for 5–7 topics per session.
